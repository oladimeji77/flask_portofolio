# flask_portofolio
Portofolio built with Python flask Framework
Ensure you have Python installed on your machine

you can set up a virtual environment to easily manage your projects **py -m venv env**

Install the libraries contained in requirements.txt **pip install -r requirements.txt**

run and test the app as is python **basic.py **

You can modify the basic.py file to suite your purpose.

Happy coding
